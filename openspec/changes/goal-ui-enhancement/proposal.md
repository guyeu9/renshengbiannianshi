# 目标模块UI优化与功能完善提案

## Context
用户需要对目标模块进行UI优化和功能完善，包括：
1. 目标主界面顶部布局调整
2. 目标详情页关联记忆改为真实数据
3. 新建全部记忆关联界面
4. 修复无任务时的灰色异常
5. 任务完成动画与交互增强
6. 更新开发文档和用户指南

## Scope

### 涉及文件
1. `life_chronicle/lib/features/goal/presentation/goal_page.dart` - 主要修改文件
2. `更新日志.md` - 更新日志
3. `开发设计文档.md` - 开发设计文档
4. 新建 `GoalAllLinksPage` 页面

### 不涉及
- 数据库表结构修改
- 其他模块的重大变更

## Behavior

### 任务1：目标主界面顶部布局调整
- **输入**：无
- **输出**：
  - 删除"年度目标"文字
  - 将年份选择器移至该位置
  - 保持与"历史年度总结"平齐
  - 字体大小调整为20px，FontWeight.w700
- **UI变更**：年份选择器位于顶部右侧，与"历史年度总结"按钮在同一行

### 任务2：目标详情页关联记忆改为真实数据
- **输入**：目标记录ID
- **输出**：
  - 从entity_links表读取真实关联数据
  - 支持小卡片显示每个关联数据
  - 保持现有UI样式（_MemoryCard和_TextMemoryCard）

### 任务3：新建全部记忆关联界面
- **输入**：目标记录
- **输出**：
  - 新建GoalAllLinksPage页面
  - 以卡片形式展示所有关联数据
  - 点击卡片跳转到对应详情页
  - 符合根目录UI设计规范

### 任务4：修复无任务时的灰色异常
- **输入**：无任务的目标详情页
- **输出**：
  - 目标详情页在无任务时显示正常
  - 其他模块类似问题也修复完成

### 任务5：任务完成动画与交互增强（P1）
- **输入**：用户勾选完成任务
- **输出**：
  - 播放全屏彩带动画
  - 轻微震动反馈（HapticFeedback.lightImpact）
  - 可选的完成音效
  - 卡片点击缩放+阴影变化动画
  - 开关切换平滑颜色过渡
  - 进度条平滑宽度变化
  - 百分比数字跳动计数动画

### 任务6：更新开发文档和用户指南（P2）
- **输入**：所有修改完成
- **输出**：
  - 更新根目录下的开发设计文档
  - 更新用户指南
  - 更新更新日志

## Risks

### 低风险
1. **顶部布局调整**：仅涉及UI布局变化，不影响功能逻辑
2. **关联数据显示**：参考美食详情页实现，有成熟模式可借鉴

### 中风险
1. **彩带动画**：需要引入新依赖，可能与现有依赖冲突
2. **无任务灰色异常**：需要排查多个模块，可能遗漏一些地方

### 缓解措施
1. 引入新依赖前先检查pubspec.yaml现有依赖
2. 逐一检查所有模块的类似问题，确保全面修复
