## 三、遗漏问题详细分析

### SC-1: API Key 明文存储

**问题位置**: `tables.dart:269`

**当前代码**:
```dart
TextColumn get apiKey => text()();
```

**问题分析**:
- API Key 以明文形式存储在数据库中
- 如果数据库文件泄露，API Key 会被暴露
- 违反了敏感信息存储安全最佳实践

**建议修复方案**:
```dart
// 方案1: 使用加密列（需要数据库支持）
TextColumn get apiKeyEncrypted => text()();
TextColumn get apiKeyIV => text()();

// 方案2: 使用 flutter_secure_storage 单独存储
class ApiKeyStorage {
  static const _storage = FlutterSecureStorage();
  
  static Future<void> save(String providerId, String apiKey) async {
    await _storage.write(key: 'api_key_$providerId', value: apiKey);
  }
}
```

**风险评估**: 高 - 敏感信息泄露

### SC-2: 类型转换 `as!` 风险

**问题位置**: 多个文件

**统计结果**: 37 处类型转换使用 `as!`

**问题分析**: 
- `as!` 是强制类型转换，如果类型不匹配会抛出 `TypeError`
- 在数据来自数据库或网络的情况下，可能存在类型不确定的数据

**修复状态**: ✅ 已修复高风险类型转换

**已修复文件**:
- `lib/core/services/providers/openai_compatible_service.dart` - AI 服务响应解析
- `lib/core/services/providers/openai_compatible_service.dart` - 嵌入服务响应解析

**修复方案**:
```dart
// 修复前
final data = jsonDecode(response.body) as Map<String, dynamic>;
final choices = data['choices'] as List?;
final content = (choices.first as Map<String, dynamic>)['message']['content'] as String;

// 修复后
final data = jsonDecode(response.body);
if (data is! Map<String, dynamic>) {
  throw Exception('AI服务返回格式错误');
}
final choices = data['choices'];
if (choices is! List || choices.isEmpty) {
  throw Exception('AI服务返回空响应');
}
final firstChoice = choices.first;
if (firstChoice is! Map<String, dynamic>) {
  throw Exception('AI服务返回格式错误');
}

```
