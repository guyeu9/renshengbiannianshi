# 人生编年史APP开发计划

## 一、技术选型

**框架**: React Native + Expo (最新版本)
- 官方推荐的移动开发方案
- 支持iOS/Android双平台
- 丰富的生态系统，便于实现地图、AI等功能

**状态管理**: React Context + useReducer (或 Zustand)
**数据库**: SQLite (本地存储) + 增量备份机制
**地图**: react-native-maps (味觉地图、旅行足迹)
**UI组件**: React Native Paper (Material Design 3)

## 三、数据模型设计

### 核心实体
1. **美食实体**: id, images, tags, rating, price, location, date, linkedFriends, linkedTravels, linkedMoods
2. **小确幸实体**: id, images, video, mood, date, location, linkedFriends, linkedTravels, linkedGoals
3. **旅行实体**: id, title, date, location, images, linkedFriends, linkedFoods, linkedMoments
4. **目标实体**: id, content, level, progress, date, linkedMoments, linkedTravels, linkedFoods
5. **朋友实体**: id, name, avatar, birthday, contact, knowDate, tags, linkedFoods, linkedTravels, linkedMoments
6. **日程实体**: 自动聚合以上所有实体

### 关联系统
- 手动关联: 用户发布时勾选
- 自动关联: 同日期/同位置/同朋友等规则
